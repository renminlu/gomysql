--------------------------------------------------------------------------------
功能：
	获取[表结构]相关信息

--------------------------------------------------------------------------------
表结构字段：
	[Field Type Null Key Default Extra]
		Field   字段名
		Type    字段类型
		Null    是否准许为空
		Key     索引属性：空：无索引、PRI：主键、UNI唯一、MUL普通索引
		Default 默认值
		Extra

--------------------------------------------------------------------------------
依赖：
	_ "github.com/go-sql-driver/mysql"  mysql驱动
	"github.com/jmoiron/sqlx" sql库的二开库
